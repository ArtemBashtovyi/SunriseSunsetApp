# SunriseSunsetApp

App requires : 
- Internet access
- Location access
- Play services

Usage : 
- Google Places API
- Sunrise Sunset API
- Retrofit2.0(api calls) 
- OkHttp
- Moxy(MVP) // Can be replaced by Loader(presenter storing)
- Leak Canary(easy tool for finding leaks) + Android Profiler(to find origin of leaks)
